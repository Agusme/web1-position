Posicionamiento con CSS

La propiedad position de CSS especifica cómo un elemento es posicionado en el documento. Las propiedades top, right, bottom, y left determinan la ubicación final de los elementos posicionados.

Un elemento posicionado es un elemento cuyo valor computado de position es relative, absolute, fixed, o sticky. (En otras palabras, cualquiera excepto static).

Uso comunes de posicionamiento en CSS

  • Relative: Se usa para realizar ajustes finos y como base para el posicionaimiento absolute.
  • Absolute: Se utiliza para superposicion de elementos, por ejemplo: cuantedo tenemos una imagen y sobre ella texto
  • Fixed: Se usa para fijar elementos en la ventana del navegador, como los navbar, algun chatbot, etc.
  • Sticky:Elementos que se pegan al contenedor al desplazarse, como menus laterales.

Posicionamiento Fixed

El elemento con posicionamiento fijo se posiciona en relación a la ventana del navegador y permanece fijo cuando se desplaza la página.

Posicionamiento Sticky

El elemento con posicionamiento sticky se comporta como un elemento relativo hasta que se desplaza a su posición límite, donde se "pega" y permanece fijo.
